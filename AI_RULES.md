# AI_RULES.md

## LLM/AI Usage Policy
- Use LLMs to assist with code generation, refactoring, debugging, and documentation.
- Always update context files when making architecture or implementation changes.
- Avoid generating code that breaks project conventions or memory.
- Use prompts from PROMPTS/ for consistent engineering workflows.

## Prompt Engineering
- Maintain reusable prompts for backend, frontend, db, debugging, refactoring, and testing in PROMPTS/.
- Prompts must be updated as workflows evolve.

## Update Policy
Update this file with every new AI workflow, rule, or prompt addition.
